Política de Privacidad

Última actualización:

Esta aplicación (el “Servicio”) utiliza las APIs de Meta (Facebook e Instagram) para recibir y responder mensajes enviados por usuarios a cuentas comerciales conectadas.

Los datos que podemos procesar incluyen contenido del mensaje, identificadores técnicos y metadatos mínimos necesarios para responder (por ejemplo, sender_id y timestamp).

Finalidad

  • Responder consultas automáticas por chat (FAQ) y brindar atención básica.
  • Mejorar el funcionamiento del bot y la calidad de las respuestas.

Almacenamiento y compartición

  • No vendemos datos personales.
  • No compartimos datos con terceros con fines de marketing.
  • Podemos registrar eventos técnicos para depuración y operación del servicio.

Seguridad

Aplicamos medidas razonables para proteger la información. Sin embargo, ningún sistema es 100% seguro.

Conservación

Los datos se conservan únicamente el tiempo necesario para operar el servicio y resolver incidencias, y luego se eliminan o anonimizan cuando corresponde.
